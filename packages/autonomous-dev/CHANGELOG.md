# Changelog

## [1.0.0] - 2026-04-01

### Added
- Initial release
- Dual-mode: CLI (ac.sh) and Agent (orchestrate.sh)
- Multi-stack support: Node.js, Python, Go, Rust, Flutter, JVM, Elixir, Swift, C++
- 3-tier quality gates with baseline diffing
- 7 AI agent templates: researcher, planner, coder, reviewer, final-review, phase-supervisor, code-simplify
- Qdrant vector indexer for semantic code search
- Claude Code hooks for enforcement
- Crash recovery via state.json
